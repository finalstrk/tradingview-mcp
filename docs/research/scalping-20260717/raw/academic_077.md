# Machine-Learning Intraday Prediction: Accuracy After Transaction Costs

**Topic ID:** academic_077  
**Family:** academic  
**Date:** 2026-07-17  
**Question:** Machine-learning intraday prediction papers that include realistic transaction costs: what accuracy survives?

---

結論：現実的コスト後に残る予測力は小さい。確認できた取引勝率は概ね40～55%で、単純な「予測方向へ毎バー売買」はほぼ全滅する。生き残る例は、予測値がコストの2倍以上あるときだけ取引する低回転型だった。

1. SPY構成銘柄リターン→SPY予測（Huddleston–Liu–Stentoft）

市場・足：SPY、5分足。入力は直前5分のSPY＋当時のS&P 500構成銘柄約500銘柄のリターン。エントリー：次の5分リターン予測の絶対値が同時点のNBBOスプレッドを超えた場合、正ならロング、負ならショート。出口：次の5分終了時。ストップなし。9:35～15:55、寄付・引けオークション除外。

1993–2016の逐次OOS（1か月検証＋1か月テスト、約78観測/日）。予測OOS \(R^2\) は最大2.01%で、方向精度・PF・取引数は未報告。中央値アンサンブルはコスト後年率6%、Sharpe 0.98、全機会の11.0%を取引。2005–2016だけでは年率5%、Sharpe 0.68、取引率18.37%。コストは実測フルNBBOスプレッド（2001年以降中央値0.008%）だが、手数料・追加スリッページは別途未計上。[査読論文](https://academic.oup.com/jfec/article-abstract/21/2/485/6400345)／[詳細本文](https://www.researchgate.net/publication/355584493_Intraday_Market_Predictability_A_Machine_Learning_Approach)

証拠A。CRSPで構成銘柄を月次更新しており典型的な生存者バイアスは抑制。ただし2016年終了、危機期依存、モデル・特徴選択による過適合余地がある。SPYの最良約定なら生存可能性はあるが、現在の小口リテールでの再現は未証明。Pine v6：部分的。売買判定は可能だが、約500銘柄取得、RF/PCR再学習、動的構成銘柄はPineの範囲外。

2. BTCコスト閾値付きXGBoost

市場・足：Binance BTC/USDT先物、1時間足、24時間。エントリー：XGBoost予測が正ならロング候補、非正なら現金。現在ポジションとの差に対し  
\(|予測値|>2\times0.1\%\times|候補-現在|\)  
のときだけ変更するため、ロング開始・終了には0.20%以上の予測が必要。出口は同じ条件で現金化、固定ストップなし。

70,128時間、2018–2025、12か月訓練＋3か月検証＋3か月OOSを27回。10bp/変更（手数料・スプレッド・スリッページ込み想定）後、251取引、年率65.40%、Sharpe 1.09、最大DD −67.42%。27区間中利益は16、正Sharpeは14。単純符号売買は10,619取引、年率−64%。25bpでも35取引、年率34.54%だが疎すぎる。勝率・PF未報告。[論文](https://arxiv.org/pdf/2606.00060)

証拠B（詳細な新規プレプリント）。複数モデル・特徴・損失・選択基準・閾値を試しており過適合リスクは高い。B&HとのSharpe差も統計的有意でない。板、可変スプレッド、部分約定、資金調達料を未再現。ただし流動的BTCで10bp以内ならリテールでも「条件付きで plausible」。Pine：部分的。閾値ロジックと60分集約は可能だが、XGBoost再学習は不可。

3. Positional-context PPO

市場・足：金・銅・銀・白金・原油・天然ガス、豪ドル・英ポンド・スイスフラン・メキシコペソ先物、1分足。13特徴からPPOが毎分 \(-1,0,+1\) の目標ポジションを出し、次分始値で変更。反対シグナルまたは0で出口、16:31に強制決済、ストップなし。10:31～16:31 ET。

2013–2021の27ローリングOOS、取引勝率40.52～55.22%、平均保有2.69～6.83分、Sharpe 0.048～3.812。平均取引期待値は僅か0.0003～0.0025%。取引数・PF未報告。コスト0.08bp、0.16bp超で多くが不採算化。[査読版情報・本文](https://arxiv.org/pdf/2406.08013)

証拠A。ただし銘柄選定を初年度データで行うため選択バイアスは残る。期待値が0.03～0.25bpしかなく、リテールのスプレッド・手数料では大半が消える公算が高い。Pine：部分的。特徴計算と固定済みネットの推論は可能だが、学習・ローリング更新は不可。
