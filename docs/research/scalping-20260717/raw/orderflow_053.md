# orderflow_053

**topic_id**: orderflow_053  
**family**: orderflow  
**date**: 2026-07-17  
**question**: NYSE TICK / ADD breadth filters for index futures scalps: quantified evidence.

---

結論：NYSE TICK／ADDを使う指数先物スキャルプについて、Grade A相当の査読研究、またはコード・データ付き再現研究は確認できなかった。見つかった定量資料は少数で、OOS検証もほぼない。ADD単独フィルターには、採用に値する定量証拠が見当たらない。

1. TICK極値からのESロング反転

市場・足：ES、15分足。  
エントリー：TICKが−800または−1000を下抜けたら1枚ロング。  
退出：固定保有。最良設定は−800で27本後（6.75時間）、−1000で16本後（4時間）。ストップなし。セッション条件は明記なし（`@ES.D`使用）。  
結果：10年間、手数料$2.50/contract。−800は勝率約50%、PF約1.1。−1000は勝率47–56%、PF約1.3。−600は全保有期間で不良。対称的な+600/+800/+1000ショートも全て不良。取引数、期間年月、スリッページ、OOSは未開示。[TradeStation Labs PDF](https://ekladata.com/oxzuIt_FvEcVBSBFzLLM4pkq6Z8/nysetick.pdf)  
Grade：B。  
Pine v6：Yes。`request.security()`でTICKを取得し固定バー退出できる。ただしこれはスキャルプではなく半日保有。  
評価：30種類の退出×3閾値という多重探索で過適合リスクが高い。PF1.1は現実的摩擦で消える可能性が高い。PF1.3は保有時間が長いため存続余地はあるが、OOSなしでは取引不可。

2. TICK・RSI・VIX・EMA 1分スキャルプ

市場・足：SPYオプション、1分足（先物ではない）。  
エントリー：TICK Hull MAが遅延シグナル線をクロス、RSI(6)が50超／未満、VIXが閾値超、価格がEMA上／下。  
退出：日次ATR比率ストップ、TICK＋RSI反転、価格–TICKダイバージェンス、15:50 ET強制退出。利益目標は日次ATRの17.5%。一部期間・閾値は公開説明だけでは特定不能。  
結果：2022年4月18日–5月27日、112取引、勝率54.46%、PF1.505、最大DD24.55%、純益122.57%。手数料は「10オプション契約の概算」、スプレッド・スリッページ・OOSなし。パラメータは同期間の純益を基準に変更された。[公開TradingView戦略](https://www.tradingview.com/script/yiBwexdm-TICK-Scalping-strategy-SPY-1-min/)  
Grade：B。  
Pine v6：Partial。シグナルは移植可能だが、オプション価格損益を先物バックテストで再現できない。  
評価：露骨なインサンプル最適化と6週間の相場依存。オプションの実スプレッドを入れれば、報告エッジが残るとは考えにくい。

3. 15分ORB＋TICK確認

市場・足：ES/NQ、5分執行。  
エントリー：9:30–9:45 ETのレンジを形成し、レンジ幅÷日足ATR(14)=0.25–0.60、VIX=16–25、突破足出来高が最初の3本平均以上。5分終値がレンジ外で、ロングはTICK>+400、ショートはTICK<−400なら終値で入る。  
退出：レンジ反対側または中点±5分ATR(14)×0.25をストップ。レンジ幅1倍で半分利確・建値移動、1.5倍で残り利確、12:00 ET全決済。  
結果：2018/2019–2025と称し、ES勝率58–62%、平均RR 1.8、週5–7回。ただし取引数、PF、費用、OOS、コード、TICK追加効果のアブレーションは未開示。[方法論](https://volatilitybox.com/research/opening-range-volatility-breakout/)  
Grade：B（数値証拠自体はC相当）。  
Pine v6：Yes。  
評価：利幅が大きいため摩擦耐性は理論上高いが、数値は監査不能。TICKが効いたのかATR・VIX・出来高が効いたのか判別できない。

共通リスク：NYSE銘柄母集団、上場廃止・市場移管、TICK/ADD算出・データフィード差、先物ロールによる非定常性がある。現時点の合理的判断は「TICK/ADDは候補フィルターだが、定量的に確立したスキャルプ・エッジではない」。特にADDについては、具体的ルールと費用込みOOSを備えた肯定証拠は実質ゼロである。
