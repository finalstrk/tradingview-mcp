# Market-Microstructure Scalping Research

**Topic ID:** orderflow_060  
**Family:** orderflow  
**Date:** 2026-07-17  
**Question:** Market-microstructure academic papers on short-horizon (minutes) return predictability with concrete evidence and Pine v6 implementability.

---

結論：分足OHLCVだけで再現でき、現代の小売コスト控除後も強いと断定できるスキャルピング手法は見つからない。学術研究の多くは Sharpe または予測精度を報告する一方、勝率・Profit Factor・約定可能性を報告していない。

1. **SPY終盤モメンタム — Grade A**

市場・足：SPY、30分（5/15分足から集計可能）。  
エントリー：前日終値から10:00 ETまでのリターンが正なら15:30に買い、負なら売り。  
決済：16:00。ストップなし。通常日も取引するが、高ボラ・高出来高・景気後退・重要指標日は強い。  
証拠：1993–2013、取引500件未満の日を除外。平均年率6.67%、年率標準偏差6.19%、Sharpe 1.08、OOS (R^2=1.4%)。論文は合理的コスト控除後も有意とするが、勝率・PF・取引数は未報告。[JFE論文](https://www.sciencedirect.com/science/article/pii/S0304405X18301351)  
Pine v6：**Yes**。  
評価：SPYは低スプレッドかつ1往復/日なので歴史的には生存可能。ただし古い標本であり、2021–2026の1,370日を用いた非学術的再検証では SPY が反転、(t=-2.3)、粗収益−0.55bp/日との報告もある。[再検証](https://www.vortexcapitalgroup.com/insights/the-1pm-echo-treasury-auctions-are-the-last-living-signal-on-the-half-hour-clock) 現在のエッジは混合的。ETFの事後選択にも注意。

2. **構成銘柄リターンによる次5分SPY予測 — Grade B**

エントリー：S&P 500構成銘柄の遅行5分リターンから次のSPYリターンをLasso、RF、アンサンブル等で予測し、(|予測値|>現在のNBBOスプレッド) の場合だけ予測方向へ売買。  
決済：次の5分終了。ストップなし。昼間、高ボラ・低流動性局面で強い。  
証拠：1993–2016の拡張窓OOS。全期間のコスト後アンサンブルSharpe 0.98。2005–2016ではPCR/RFが年率9%/8%、Sharpe 0.88/0.81。勝率・PF・正確な取引数は未報告。[SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3726765)  
Pine：**Partial**。多数銘柄、再学習、NBBO履歴が必要。  
評価：結果の大半は十進化前。スプレッドは控除するが、遅延・市場インパクト・手数料を十分再現しておらず、小売での生存は不確実。構成銘柄履歴の扱い次第で生存者バイアスも生じる。

3. **1分急落後リバーサル — Grade A**

市場：NASDAQ-100、1分。  
エントリー：1分リターンが推定平均−3.09標準偏差以下で、二つの異常判定法がともに0.1%裾事象と認定した直後に買う。  
決済：次の1分終了。ストップ・セッションフィルターなし。  
証拠：2014–2019、全NASDAQ-100銘柄。急落幅の平均31%が次の1分で反転し、大型・高流動性上位20%で強い。ただしイベント数、勝率、PF、P&L、コスト、OOSは未報告。[QREF論文](https://doi.org/10.1016/j.qref.2021.05.004)  
Pine：**Yes**（近似）。  
評価：31%は勝率ではない。ショック時のスプレッド拡大とスリッページが最大になるため、成行では消滅する可能性が高い。指数構成銘柄選択と閾値探索にもバイアス懸念。

4. **最良気配のキュー不均衡 — Grade A**

市場：NASDAQ大型株、イベント時間。  
シグナル：(I=(Q_{bid}-Q_{ask})/(Q_{bid}+Q_{ask}))。論文は売買ルールを定義せず、機械化するならロジット確率が50%超で買い、未満で売り、次のミッド価格変化で決済。ストップなし。10:00–15:30。  
証拠：10銘柄、2014年252日。OOS AUCは大型ティック株0.752–0.805、小型ティック株0.581–0.642。損益・勝率・PF・コストなし。[論文](https://arxiv.org/abs/1512.03492)  
Pine：**No**。履歴板数量、キュー順位、イベント時刻が取得不能。  
評価：次ティック方向の予測力は本物だが、スプレッドを跨ぐ利益とは別物。流動性の高い生存銘柄だけを選んでおり、 retail execution で残る証拠はない。

補強的な否定結果として、MNQの5分足947日・14戦略群をwalk-forward検証した研究では、粗エッジ0.07–1.50ポイント/取引に対し往復コスト2ポイントで、全基準を満たす戦略はゼロだった。[arXiv](https://arxiv.org/abs/2605.04004) 現状の最有力候補はSPY終盤モメンタムだが、最新期間で独立再検証すべきであり、他三つは予測現象であって実証済み小売スキャルプではない。
