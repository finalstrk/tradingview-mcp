# Research: Momentum/MACD Divergence Scalps

**Topic ID:** pullback_035  
**Family:** pullback  
**Date:** 2026-07-17  
**Question:** Momentum/MACD divergence scalps: does any quantified evidence support them?

---

## Analysis

結論：MACD／モメンタム・ダイバージェンスを1–15分足でスキャルピングすることを支持する高品質な証拠は見つからない。比較的強い証拠があるのは、ダイバージェンスではなく、時間帯限定のイントラデイ・モメンタムである。

1. Triple-confirmed divergence — Evidence C

- 市場／足：NASDAQ-100現構成銘柄、15分足、IBKRデータ、1年。
- エントリー：価格が新安値／高値を付け、同一ピボットでRSI・Stochastic・MACDが三つとも逆方向に推移。詳細なピボット幅、約定タイミングは非開示。
- 出口：ATRトレーリングとされるが、倍率、初期ストップ、時間切れ、セッション条件は非開示。
- 数値：1,116取引、Sharpe 2.99。勝率、PF、コスト、OOSは公開要約にない。別の5年・日足テスト（54銘柄、勝率約37%、PF 3.3）は15分足の検証にならない。[Quant Market Lab](https://quantmarketlab.com/updates/)
- Pine v6：部分的。ピボット確定後なら実装可能だが、欠落パラメータの補完は別戦略を作ることになる。
- 評価：現在のNASDAQ-100銘柄だけなら重大な生存者バイアス。単年、三重フィルター、出口選択にも過学習リスク。コスト非開示のため小売スプレッド後の存続は判断不能。

2. MNQ 5分足 volume-spike momentum — Evidence B（強い否定証拠）

- エントリー：出来高が直近20本に対する上位10%に入り、そのバーが上昇なら次バー始値で買い、下落なら売り。
- 出口：次の5分バー終了時。RTH 09:30–16:00 ET。
- 数値：2021–25、947日、72,604本。上昇2,119取引：勝率51.6%、平均純損益−1.94pt、t=0.07。下落2,409取引：50.1%、−2.50pt、t=−0.64。2pt往復コスト込み。2022→2023、2022–23→2024、2022–24→2025の拡張OOS。[論文](https://arxiv.org/abs/2605.04004)
- Pine v6：Yes。
- 評価：PFは非報告だが純損失。連続先物のロール処理と短い市場期間は弱点。ただし生存者バイアスは小さく、現実的コスト後にエッジなしという結論はかなり強い。

3. SPY Noise-Area momentum — Evidence B

- エントリー：過去14日、同時刻までの始値からの絶対変動平均で上下バンドを作り、ギャップ調整。HH:00／HH:30に上抜けなら買い、下抜けなら売り。
- 出口：買いはmax（上バンド、当日VWAP）割れ、売りはmin（下バンド、VWAP）超えを30分ごとに判定。16:00全決済。
- 数値：2007–24、1分足、7,668取引、勝率37%、平均$0.09／株。動的最大4倍レバレッジ版は年率19.6%、Sharpe 1.33、MDD 25%。手数料$0.0035＋スリッページ$0.001／株。[SSRN/PDF](https://concretumgroup.com/wp-content/uploads/2026/02/Beat-the-Market.pdf)
- Pine v6：Yes、1–15分足から30分判定可能。
- 評価：SPY単体なので生存者問題は小さいが、OOSなし、段階的な出口・レバレッジ改良は過学習懸念。想定スリッページが実際のスプレッドより楽観的で、優位性は弱まる。

4. KOSPI last-30-minute momentum — Evidence A

- エントリー：overnight、2番目、11番目の30分リターンが全て正なら14:30買い、全て負なら売り。
- 出口：15:00決済。ストップなし。
- 数値：2004–16、3,087日。両建て条件772取引、成功率58.8%、収益16.77%、Sharpe 0.167。買いのみ482取引、66.81%、27.2%、Sharpe 0.291。推定実効スプレッド後も頑健と報告。[査読論文](https://www.mdpi.com/1911-8074/15/11/523)
- Pine v6：部分的。5–15分足から30分窓を合成可能だが、KOSPI現物指数は直接売買不能。
- 評価：OOSなし、単一市場、指数構成変更の影響あり。ETF／先物の実コストを使っていないため、小売での存続は未証明。

総合すると、ダイバージェンス固有のエッジは弱く、コスト込みOOS証拠は実質ゼロ。検証価値があるのは、MACDダイバージェンスよりSPY型の時間帯別モメンタムである。
